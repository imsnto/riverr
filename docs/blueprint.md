# **App Name**: TimeFlow

## Core Features:

- Centralized Dashboard: Dashboard with project & task overview, time logs, and quick actions.
- Manual Time Logging: Manual time entry with project, task, start/end times, and notes.
- Timer-Based Tracking: Start/stop timer for task tracking with live updates.
- Role-Based Access: User authentication and role-based access control (Admin/Member).
- Task Board: Kanban-style task management with drag-and-drop functionality.
- Meeting Project Suggestion: Suggest project assignments for unmapped Slack Meeting Logs, based on channel name, admin mappings, using fuzzy match tool. Display dropdown for confirmation.
- Timesheet Export: View of all team member timesheets for admins

## Style Guidelines:

- Primary color: A subdued teal (#46A5A5) evokes feelings of reliability and precision.
- Background color: A very light teal (#F0FAFA), visibly close to the primary in hue, creates a calm background.
- Accent color: Muted yellow (#A5A046) provides contrast and draws attention to key actions.
- Body and headline font: 'Inter', a sans-serif, will provide a clean, neutral, machined aesthetic suitable for UI elements and informational displays.
- Clean, card-based layout with clear sections for projects, tasks, and time logs.
- Use minimalist, consistent icons for common actions and states.
- Subtle transitions and animations for task updates and timer interactions.
